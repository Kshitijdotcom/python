# PhotoCrest Backup Files

This directory contains backup copies of the main application files.

## Files

- `index_backup.html` - Frontend backup (HTML/CSS/JavaScript)
- `enhancement_engine_backup.py` - AI enhancement engine backup
- `image_editor_server_backup.py` - Flask server backup
- `models_backup.py` - Database models backup

## Usage

To restore from backup:
```bash
copy backup\index_backup.html index.html
copy backup\enhancement_engine_backup.py enhancement_engine.py
copy backup\image_editor_server_backup.py image_editor_server.py
copy backup\models_backup.py models.py
```

These backup files are updated whenever significant changes are made to the main application.
